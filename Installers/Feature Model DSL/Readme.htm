Feature Modeling DSL

Installation

For installation of the Feature Modeling DSL simply double-click on the FeatureModelDSL.msi, which you can find in the current directory and follow the on-screen instructions.

Getting started

  • Install the Feature Modeling DSL.
  • Open Visual Studio 2005.
  • Open any existing or create a new project or solution.
  • Right click on a project in the Solution Explorer and select Add->New Item
  • Select Visual C# Project Items and select the FeatureModel and fill in the name for the model you want to create.

  • After adding the new item an empty document will open and in the Toolbox the concepts provided by the Feature Modeling DSL become available

Elements of a Feature Model

In order to create your feature model simply drag the elements from the toolbox to the document. With a right click on the document surface you can check the correctness of the diagram  Errors will be shown in the output window. Furthermore you can attach conditions, constraints and a description to each feature by double clicking on the feature in the feature model. Therefore we can create the requirements documents from the feature model. Remember, requirements consists of capability, constraint and condition.

In addition you can generate an xml description from the model by right click on the document surface. This xml description is intended to be the input for the feature model configurator.

The different elements are explained below.

Root Feature

In each feature model there is exactly one root feature that defines the root node of the feature model.

Features

As mentioned earlier with a double click on the feature you can add conditions, constraints and a descirption to each feature. In addition you can choose the type of feature as either mendatory, optional or feature set feature.

Mandatory Feature

Mandatory features are shown as features with a decorator that shows a filled circle. These features are common to all products in the product line.

Optional Feature

Optional features are features that may or may not be part of a specific product. In the model these features are shown with a white circle decorator.

Feature Set

A feature set defines the minimum and maximum number of features that can to be selected within this group of features. In the example below zero to two features need to be selected for a valid model.
