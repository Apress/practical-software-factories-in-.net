Feature Modeling Configuration

Installation

For installation of the Feature Configurator simply-double click on the FeatureModelConfigurator.msi, which you can find in the current directory and follow the on-screen instructions.

Getting started

  • Install the Feature Modeling Configurator.
  • Open the Feature Model Configurator.
  • Select File->open.
  • Select a feature model that you saved using the feature model DSL.

  • Select or unselect the variable features that are indicated by the "?" in the check box.
  • Save the configuration of the features for a certain product.
  • The Status window will validate your choices and make sure the model is complete.
  • Alternatively you can reset or close the model.
